
 Building thymeleaf-extras-conditionalcomments
 ---------------------------------------------
 
 To build thymeleaf-extras-conditionalcomments you will need Maven 2. You can
 get it at:
 
     http://maven.apache.org

 Build and install the project executing, from the project's root folder:
 
     mvn clean:clean install
     
 And you will get a fresh target/thymeleaf-extras-conditionalcomments-{version}.jar
 file. You will also get it installed in your local repository at:
 
     $M2_REPO/org/thymeleaf/extras/thymeleaf-extras-conditionalcomments/{version}/thymeleaf-extras-conditionalcomments-{version}.jar



 Generating Javadoc for thymeleaf-extras-conditionalcomments
 -----------------------------------------------------------
 
 If you wish to generate the javadoc for thymeleaf-extras-conditionalcomments, 
 execute this from the project root folder:
 
     mvn javadoc:javadoc
     
 This will generate the javadoc documentation in HTML format in:
 
     target/site/apidocs
